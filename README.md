## Setup
To run this project, install it locally using npm:

```
clone with ssh
$ git clonegit@github.com:herenacreative/saku-saku-react-js.git
or clone with https
$ git clone https://github.com/herenacreative/saku-saku-react-js.git
$ cd saku-saku-react-native
$ npm install
$ npm start
```
